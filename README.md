# Ecommerce Web Application

A full-stack **Ecommerce Web Application** built with a **React (Vite) frontend** and a **Node.js/Express backend**.  
This project provides basic ecommerce functionality including product display, routing, shopping cart, and REST APIs connected to a MySQL database.

## Tech Stack

**Frontend**                                       
✔ React (Vite)  
✔ Material UI  
✔ Axios for API requests  
✔ React Router (routing)  
✔ Font Awesome icons

**Backend**                                                
✔ Node.js + Express  
✔ MySQL database  
✔ CORS & body-parser middleware  
✔ dotenv for environment configuration

## Features                                                         

✨ Product browsing & list view  
✨ Product details page  
✨ Add to cart functionality  
✨ RESTful API endpoints (CRUD) for products  
✨ Separate frontend & backend setup  
✨ Hot reload on backend using **nodemon**

⚠️ *This is a base project — you can extend with user auth, order placement, payment integration, reviews, filtering, etc.*
